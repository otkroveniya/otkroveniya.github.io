Официальный сайт http://tox.chat/
Наиболее продвинутые клиенты:
Windows http://isotoxin.im/
Android http://play.google.com/store/apps/details?id=com.zoffcc.applications.trifa

Другие TOX клиенты, под разные ОС включая мобильные версии:
http://wiki.tox.chat/clients и http://tox.chat/download.html
Сервис сокращения ID http://toxme.io/
Русскоязычные TOX чаты http://toxme.io/u/kalina (добавление через бота)

I2P


Децентрализованя анонимная сеть для доступа к .i2p сайтам и другим сервисам.

Java версия http://geti2p.net/ru/
С++ версия http://i2pd.website/, http://github.com/PurpleI2P/i2pd/releases


Запуск I2P в портативном режиме:
Z:\soft\i2p\I2Psvc.exe -c Z:\soft\i2p\wrapper.config

В wrapper.config заменить пути к файлам и папкам: 
set.JAVA_HOME=Z:/soft/jre
wrapper.java.command=Z:/soft/jre/bin
wrapper.java.command=Z:/soft/jre/bin
set.JAVA_HOME=Z:/soft/jre
wrapper.java.command=%JAVA_HOME%/bin/java.exe
wrapper.java.classpath.1=Z:/soft/i2p/lib/*.jar
wrapper.java.library.path.1=Z:/soft/i2p
wrapper.java.library.path.2=Z:/soft/i2p/lib
wrapper.java.additional.2=-Di2p.dir.base="Z:/soft/i2p"
#wrapper.java.additional.3=-Di2p.dir.pid="Z:/soft/i2p"
#wrapper.java.additional.4=-Di2p.dir.temp="Z:/soft/i2p"
#wrapper.java.additional.5=-Di2p.dir.config="Z:/soft/i2p"
wrapper.logfile=Z:/soft/i2p/wrapper.log
wrapper.syslog.logfile=Z:/soft/i2p/syslog.log
